Commodity Price Tracker Crude Oil Natural Gas

The Commodities Provider Application allows users to keep track of the approximate value of energy commoditites such as crude oil and natural gas.

Currently, the Application only provides commodities value for Brent Crude Oil for the last five days. I will be working on expanding these features to allow users to select the commodity they are interested in and set a date range for results
